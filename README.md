# ✈️ HighCloud Airlines Analysis Dashboard

## 📖 Project Overview

The **HighCloud Airlines Analysis Dashboard** is an end-to-end Business Intelligence project built using **SQL, Excel, Power BI, and Tableau**. This project analyzes airline operations, passenger traffic, route performance, and operational KPIs to generate actionable business insights.

The dashboard helps stakeholders monitor airline performance, identify high-performing routes, analyze passenger travel patterns, and optimize operational efficiency through interactive visualizations.

---

## 🎯 Business Objective

Airline companies generate large volumes of operational data, making it challenging to track performance and identify growth opportunities.

This project aims to:

✅ Analyze passenger transportation trends  
✅ Identify top-performing airlines and routes  
✅ Compare weekday vs weekend travel behavior  
✅ Evaluate load factor and operational efficiency  
✅ Monitor key airline KPIs through interactive dashboards
